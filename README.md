# 📖 About

Archive is a minimal Pinterest-inspired gallery designed to preserve moments instead of simply storing photos.

Every card tells a small story—from coffee dates and road trips to random nights, books, cafés, and memories.

The focus of the project was to build something that feels warm, calm, and personal while recreating the masonry aesthetic of Pinterest.

---

# ✨ Features

- 📌 Pinterest-inspired masonry layout
- 🖼 Beautiful responsive image cards
- 🌙 Elegant light & dark mode
- ❤️ Favorite button on every memory
- 🏷 Category tags
- 🔍 Search bar
- 📱 Fully responsive design
- ⚡ Smooth animations
- 🎨 Clean typography and minimal UI

# 📸 Design Philosophy

Instead of creating another image gallery, Archive was built around one idea:

> **"Some memories deserve more than just a folder."**

The interface intentionally prioritizes whitespace, large imagery, elegant typography, and storytelling to make browsing memories feel calm and immersive.
